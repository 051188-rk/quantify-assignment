# Stock Market Dashboard (MERN Stack)

A real-time stock market dashboard built with MongoDB, Express.js, React, and Node.js. This application allows users to track stock prices, view historical data with interactive charts, and manage a personal watchlist. The app integrates with financial data APIs to provide up-to-date market information with a clean, responsive UI.

## Features
- User authentication (register/login)
- Real-time stock price tracking
- Interactive stock charts with historical data
- Personalized watchlist management
- Responsive design for all devices
